## How To Build

### prerequisite

1. [Install Docker](https://docs.docker.com/engine/install/)
2. [Install Docker Compose](https://docs.docker.com/compose/install/)

### Build Steps

#### 1. Build Docker image

```sh
gh repo clone Elkhaberi/wordpress-docker-compose
```

```sh
docker-compose up -d
```

```sh
docker-compose ps
```
